# OTel Python Demo

Learning Observability concepts with OpenTelemetry in Python using a Flask application.

This repository is a progressive learning space where Observability with OpenTelemetry concepts are explored step by step 
    — starting from basic trace setup and building toward a well-demonstrated observability demo. 
Each commit reflects a new concept learned and applied hands-on.

Status

Work in progress 
    — concepts are added incrementally as they are learned.
